# Ledger AI Stitch exports

The design exports are deliberately separated by target platform:

- [`web-desktop/`](web-desktop/README.md): desktop web screens and the shared Ledger AI design-system asset.
- [`mobile-responsive/`](mobile-responsive/README.md): mobile application and responsive-web screens.

Before implementing either target, read the shared
[`IMPLEMENTATION.md`](IMPLEMENTATION.md) guide. It defines source precedence,
semantic tokens, responsive behavior, restrained glass usage, reusable module
boundaries, backend readiness, and the recommended delivery order.
